# Palindrome Checker

A responsive **Palindrome Checker** web application built using **HTML, CSS, and JavaScript**. This project allows users to enter a word, phrase, or number and instantly checks if it reads the same backward as forward.

##  Features

- Real-time palindrome checking
- Supports words, phrases, and numbers
- Ignores spaces, punctuation, and capitalization
- User-friendly and responsive design
- Instant feedback on input validity

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

##  How It Works

1. Users enter a word, phrase, or number in the input field.
2. JavaScript processes the input by removing spaces and punctuation, and converting to lowercase.
3. The script checks if the processed input reads the same backward as forward.
4. The result is displayed instantly on the page.

##  How to Use

1. Clone the repository:
   ```
   git clone https://github.com/Fida-ur-Rahman/Palindrome-Checker.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
4. Enter a word, phrase, or number to see if it is a palindrome.
